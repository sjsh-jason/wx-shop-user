## API 接口清单

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 促销商品列表 | GET | /api/promotion/products | 获取促销商品列表 |
| 商品详情 | GET | /api/promotion/products/:id | 获取商品详情 |
| 创建预约 | POST | /api/reservation | 抢购预约 |
| 我的预约 | GET | /api/reservation/my | 获取我的预约列表 |
| 预约详情 | GET | /api/reservation/:id | 获取预约详情 |
| 核销预约 | POST | /api/reservation/:id/verify | 核销预约 |
| 管理商品(商家) | GET/POST/PUT/DELETE | /api/admin/promotion/products | 商家端商品管理 |
