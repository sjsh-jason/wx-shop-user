## 数据库表设计

### user（会员表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| openid | VARCHAR(64) | 微信 openid，唯一 |
| nickname | VARCHAR(64) | 昵称 |
| avatar | VARCHAR(255) | 头像URL |
| phone | VARCHAR(20) | 手机号 |
| points | INT | 积分余额 |
| level | TINYINT | 会员等级 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
