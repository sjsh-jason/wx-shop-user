## API 接口清单

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 奖品列表 | GET | /api/prizes | 获取抽奖奖品列表 |
| 奖品详情 | GET | /api/prizes/:id | 获取奖品详情 |
| 执行抽奖 | POST | /api/lucky-draw | 用户抽奖 |
| 我的抽奖记录 | GET | /api/lucky-draw/my | 获取我的抽奖记录 |
| 商家奖品列表 | GET | /api/merchant/prizes | 商家获取奖品列表 |
| 商家创建奖品 | POST | /api/merchant/prizes | 商家创建奖品 |
| 商家更新奖品 | PUT | /api/merchant/prizes/:id | 商家更新奖品 |
| 商家客户列表 | GET | /api/merchant/customers | 商家获取客户列表 |
