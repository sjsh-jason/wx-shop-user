# 餐饮会员小程序 - 需求文档

## 项目概述

- **项目名称**：餐饮会员小程序
- **项目类型**：微信小程序（原生开发）
- **目标用户**：快餐/外卖餐饮企业的老客户
- **核心目标**：留住老客户，提升用户复购率
- **参考原型**：餐饮会员小程序.html

---

## 一、客户端（C端用户）功能

### 1.1 扫码注册登录
- 小程序提供专属会员二维码
- 新用户扫描二维码 → 进入小程序 → 微信授权注册/登录 → 自动成为该商家的会员
- 老用户扫描二维码 → 快速登录进入会员中心

### 1.2 促销产品抢购预约
- **功能描述**：
  - 查看每日促销产品列表
  - 展示促销商品图片、名称、原价、促销价、库存
  - 用户点击抢购预约促销产品
  - 预约成功后生成核销二维码
  - 商家端扫描核销后，用户按促销价线下购买
  
- **业务流程**：
  1. 用户浏览促销商品
  2. 点击抢购预约
  3. 系统确认库存
  4. 生成预约记录和核销二维码
  5. 用户到店出示二维码
  6. 商家扫码核销
  7. 用户按促销价消费

### 1.3 积分商城 & 商品兑换
- **功能描述**：
  - 展示可兑换商品列表（菜品、饮品、周边礼品等）
  - 显示商品所需积分、库存
  - 用户使用积分兑换商品
  - 兑换成功后商品进入「待领取列表」
  - 待领取商品生成核销二维码
  - 商家扫描核销后，线下提供商品

- **业务流程**：
  1. 用户浏览积分商城
  2. 选择商品点击兑换
  3. 系统检查积分余额和库存
  4. 扣减积分，生成兑换记录
  5. 商品进入待领取列表
  6. 用户到店出示二维码
  7. 商家扫码核销
  8. 用户领取商品

### 1.4 幸运抽奖
- **功能描述**：
  - 九宫格抽奖形式
  - 奖品类型：
    - 谢谢参与
    - 积分奖励（50积分、200积分、500积分等）
    - 实物商品（免费饮品、招牌菜1份等）
    - 代金券（¥10代金券、¥50大额券等）
  - 消耗积分参与抽奖（如50积分/次）
  - 展示中奖播报

### 1.5 积分规则（用户端）
- **每日签到**：用户每日签到获得积分奖励
- **消费获积分**：抢购预约核销后，按消费金额增加积分
- **兑换获积分**：积分兑换商品核销后，按商品金额增加积分
- **积分明细**：用户可查看积分收入和支出记录

## 二、商家端（B端管理）功能

### 2.0 商家权限控制（手机号白名单）
- **权限控制方式**：采用手机号白名单机制
- **白名单管理**：
  - 在数据库 `merchant_whitelist` 表中维护商家手机号
  - 只有白名单内的手机号才能进入商家端功能
  - 白名单支持启用/禁用状态管理
- **商家进入流程**：
  - 商家使用微信授权登录
  - 系统检查用户手机号是否在白名单内
  - 白名单内的用户显示商家端入口
  - 非白名单用户只能使用普通用户功能

### 2.1 扫码核销

### 2.1 扫码核销
- **核销预约记录**：
  - 商家扫描客户「购买预约」二维码
  - 显示预约商品信息、促销价格
  - 确认核销，标记预约记录为已使用
  - 用户按促销价线下消费

- **核销待领取商品**：
  - 商家扫描客户「待领取商品」二维码
  - 显示商品信息、兑换积分
  - 确认核销，标记领取记录为已使用
  - 线下提供商品给用户

### 2.2 商品中心管理（基础商品库）
- **功能描述**：建立统一的商品库，管理所有商品信息
- **商品类型**：
  - 代金券
  - 菜品
  - 其他礼品
- **商品信息**：
  - 商品基本信息（名称、图片、描述）
  - 商品类型（代金券/菜品/其他礼品）
  - 库存管理
  - 状态管理（上架/下架）
- **商品操作**：
  - 添加新商品
  - 编辑商品信息
  - 上架/下架商品
  - 查看商品列表

### 2.3 促销活动管理
- **功能描述**：基于商品库创建促销活动
- **活动配置**：
  - 从商品库选择要促销的商品
  - 设置促销价格
  - 设置活动时间（开始时间、结束时间）
  - 设置活动库存
- **活动操作**：
  - 创建新促销活动
  - 编辑活动信息
  - 启用/禁用活动
  - 查看活动销售数据

### 2.3 积分规则配置
- **每日签到积分**：设置用户每日签到可获得的积分数量
- **抢购预约积分规则**：设置消费金额与积分的比例（如消费1元获得10积分）
- **积分兑换积分规则**：设置兑换商品金额与积分的比例（如商品价值1元获得5积分）

### 2.4 幸运抽奖管理
- 配置抽奖奖品池（添加/删除奖品）
- 设置各奖品的中奖概率
- 管理抽奖活动（开启/关闭、设置参与所需积分）
- 查看中奖记录

### 2.5 客户列表管理
- 查看所有注册会员列表
- 查看客户基本信息（头像、昵称、手机号、会员等级）
- 查看客户积分余额、优惠券数量
- 查看客户消费记录和积分明细
