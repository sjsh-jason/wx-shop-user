
幸运抽奖功能优化：
1.小程序抽奖页面： 
1）抽奖首页（核心页）
UI 组件：

顶部：活动标题（固定：幸运大抽奖）+ 活动剩余时间

中间：8 格转盘（固定样式，中心显示店铺 LOGO）

转盘格子文字：按 8 格奖项配置（见 3.3）

指针：点击「开始抽奖」后旋转，最终指向中奖格子

底部：

剩余抽奖次数展示（例：今日可抽 2 次）

「开始抽奖」按钮（次数 > 0 时可点击，否则置灰）

「活动规则」入口（跳转规则详情页）

「我的中奖记录」入口（跳转记录页）

交互逻辑：

进入页面自动校验用户身份（已授权 + 会员→正常展示；未授权→弹窗引导授权；非会员→引导注册）

点击「开始抽奖」：

消耗 1 次抽奖机会

播放转盘旋转动画（时长 2-3 秒，不可手动停止）

动画结束后弹窗展示中奖结果（含奖品名称 + 使用说明）

自动发放奖品（卡券→卡包，积分→积分账户）

次数为 0 时：弹窗提示「去消费 / 签到获取更多抽奖机会」，并提供「去消费领取」「去签到」跳转入口 

2）活动规则详情页
展示内容：直接使用之前生成的「可复制到小程序的富文本规则」

样式要求：适配小程序屏幕，支持上下滚动，字体清晰（最小 14px）

3）中奖记录页
展示字段：中奖时间、奖品名称、奖品状态（未使用 / 已使用 / 已过期）、使用截止时间
